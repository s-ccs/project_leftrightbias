# A simple spatial arrangement modification to increase statistical power in concurrent two alternatives choice tasks

Code-Repository for a study on the left-right bias (and up-down bias) using eye-tracking.

- **Data** can be found in `./code/data/`, direct output of GazePoint3 recorded via OpenSesame/PyGaze
- **Experimental** code can be found in `./code/experiment/Final-Experiment.osexp`
- **Analysis** run `./code/scripts/DiagonalBias.ipynb` to generate the first results figure
- **Simulations** run `./code/nb_2025-07-18_animationLeftRight.jl` using [`Pluto.jl](https://plutojl.org/) to recover the simulation results
